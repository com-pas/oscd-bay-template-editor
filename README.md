# oscd-bay-template-editor

This project provides a plugin for OpenSCD to edit bay templates in SCL (IEC 61850-6) files.

## Usage

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```
3. Open `demo/index.html` in your browser to use the Bay Template Editor plugin within OpenSCD.

## Development

- The main plugin source is in `bay-template-editor.ts`.
- Demo and test files are in the `demo/` folder.

## Requirements

- Node.js and npm
- OpenSCD or compatible environment
